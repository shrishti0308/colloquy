# colloquy

**colloquy** is a real-time technical interview platform built for structured, signal-rich coding interviews. It combines live video, a VSCode-powered editor, secure code execution, and automated evaluation into a single interview surface.

---

## Features

- 1-on-1 **video interviews** with room locking
- **VSCode-like code editor** for real interview conditions
- **Secure, isolated code execution** with test-case based evaluation
- **Auto feedback**: deterministic pass/fail outcomes
- **Real-time chat** alongside the interview
- **Mic, camera, screen share, and recording controls**
- **Practice mode** for solo problem-solving
- **Live dashboard** with interview and system stats
- **Background jobs** for async workflows (evaluation, cleanup, notifications)
